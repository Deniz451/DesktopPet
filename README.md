# Desktop Pet

## Instalation
<ol>
<li>Download the repo</li>
<li>In cmd navigate to the repo dir and run this command</li>

<br>

```
pyinstaller --windowed main.py
```

<br>

<li>Open newly created main.spec file and under Analysis change datas to [('gif', 'gif')]</li>
<li>Run the .exe file in dist folder and enjoy your desktop pet
</ol>
